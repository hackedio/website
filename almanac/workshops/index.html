Workshops

Here is the list of workshops. All the workshops will be happening on the ground floor in the main hacking area. The are 2 corners that we will be using. This schedule will most likely still be subject to change before the event.

The Workshop Labs

Mailjet APIs
13:30 - 14:00
Florian Le Goff, Developer Evangelist @ Mailjet

Want to integrate email into your hack? This is the time to grab Florian from Mailjet and have him teach you how to integrate their APIs into your app.

Firefox OS
14:15 - 15:45
Francisco Jordano + Ruth John, Telefónica Digital

Want to make your web app work on Firefox OS? Francisco and Ruth will run you through all the possibilities and best practices in no time!

Developing for Philips Hue Lights
16:00 - 17:00
Kevin Toms, Developer Evangelist @ Philips Hue

Philips is bringing some full on development kits for their Hue lights and will be guiding you through how to get started with them. Just make sure to bring your disco outfit for when you manage to get the stroboscope script working.

Row M

Building apps for Windows 8, Windows Phone 8, and Windows Azure Mobile Services
13:30 - 14:00
Paul Lo, Technical Manager @ Microsoft UK

The Windows team will be on site to help everyone get started with their Windows (Phone) 8 apps. A series of small workshops will help everyone with the basics and provide some guidance for the pros.

Getting started with the Nokia Music API
14:00 - 14:30
Pedro Lamas, Software Engineer @ Nokia Music

Nokia's Music APIs are a collection of web service and Windows APIs that allow you to offer elements from Nokia's Music service within the look & feel of your own application. Come and learn about using the Nokia Music API in your own apps and accessing the vast Nokia Music store.

Intro to Git
15:00 - 15:30
John Stevenson, Developer Evangelist @ Heroku

Never used Git but wish you had time to learn? With a quick Git workshop you can be up an running quickly and know enough to not hang yourself.

CURL up to the Heroku APIs
15:30 - 16:00
John Stevenson, Developer Evangelist @ Heroku

Discover how easy it is to control Heroku using the new platform API. Using your favourite language or even just CURK and JSON you can quickly build something great.

Start charging money in 15 seconds
16:15 - 16:45
Cristiano Betta, Developer Evangelist @ PayPal

Taking payments can seem daunting but Cristiano Betta (PayPal) will be getting everyone set up to take payments on their site or mobile app. It's so easy he promises it's doable in 15 seconds or less.

Real-time Video Streaming Apps with OpenTok
17:00 - 18:00
Ankur Oberoi, Developer Evangelist @ TokBox

We finally have everything we need from the Web Platform to make awesome real-time collaborative apps. Come learn technologies like WebRTC and Websockets through rich APIs like OpenTok to put together a neat multiplayer game in under an hour

Row Q

Meta Broadcast
13:30 - 14:00
Jonathan Tweed, Product Manager @ Meta Broadcast

Meta Broadcast will demo their Atlas API which includes full UK TV and radio listings. Want to combine the BBCs crazy gadgets with some amazing data? This is where you start.

Discovering YQL
14:15 - 15:15
Matthew Jones, YDN Developer @ Yahoo!

Always wanted to get started with YQL, BOSS, or any of Yahoo!'s other APIs? Drop by as Matthew Jones walks you through the basics of getting setup and making your first YQL call.

Building a mobile application without writing a line of code
15:30 - 16:30
Elena Álvarez, linguist @ Apicultur

Building a mobile app used to be a task only for the brave and very skilled ones. Not anymore. In this workshop, we'll learn how to build a basic mobile application. Using tools that require no developing skills and with the power of APIs, novices will be able to make their first app painlessly.

The Guardian Content API
16:45 - 17:45
Cantlin Ashrowan, Product Owner @ The Guardian

Did you know you can build applications with the data from The Guardian? Their content API includes information sourced from all their publications including various forms of media.

We also woud like to allow anyone to run their own workshops in a third area. If you think you'd like to give a workshop then email Cristiano on cristiano@hacked.io
